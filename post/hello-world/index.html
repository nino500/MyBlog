你好，欢迎来到我的博客！

这是我的第一篇文章，我想记录一下创建这个博客的过程。

为什么写博客？

我一直想有一个自己的博客，用来记录学习笔记、技术心得和生活感悟。之前尝试过各种平台，但总是觉得不够自由。

直到我决定自己动手做一个博客网站，从前端到后端，从设计到实现，全程自己掌控。这种感觉真的很棒！

技术选型

这个博客使用了以下技术：

  • 前端：HTML + CSS + JavaScript
  • 后端：Python Flask
  • 数据存储：JSON文件
  • 部署：本地运行

选择Flask是因为它轻量、简单，非常适合个人博客这种小型项目。
